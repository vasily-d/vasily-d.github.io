Главная Стек используемых технологий Учебное портфолио

Разработка сайтов на wordpress по вашему макету с админкой, верстка лендингов, телеграм боты

  • Сайты на WordPress (с нуля или на шаблоне)
  • Админка под лендинги на Wordpress
  • Телеграм боты на php
  • Одностраничные сайты (landing page) и сайты визитки
  • Многостраничные сайты
  • Промо-сайты
  • Правки по верстке, настройка почты, почта telegram, почта WP, JS(JQUERY), калькуляторы и т.д.

Стек используемых технологий

— CMS Wordpress (посадка на WP)

— Написание шорткодов, почта wordpress и т.д.

— Разработка на PHP

— HTML5, CSS3 (верстка на основе макета)

— Bootstrap

— Адаптивная и семантическая верстка

— Оптимизация скорость загрузки сайта (Google PageSpeed) и валидация

— Adobe Photoshop, Figma, Zeplin

— Телеграм боты на php

— Telegram почта

— Js (JQuery)

— Технология Ajax

— Технология lazy loading и другие

— Технология БЭМ и Flex

— Препроцессоры Less/Sass

— Сборщик проектов Gulp

— Использую Npm, Git, Bower
