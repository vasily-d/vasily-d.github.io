Верстка лендингов, разработка сайтов с админкой (wordpress, modx) по вашему макету

Контакты: Telegram ВКонтакте

Добрый день. Я front-end разработчик. В данной сфере работаю уже более 2 лет. Занимаюсь созданием веб-сайтов начиная от верстки и заканчивая посадкой сайтов на CMS Wordpress и ModX.

Виды услуг:

  • Одностраничные сайты (landing page) и сайты визитки
  • Промо-сайты
  • Многостраничные сайты
  • Сайты на MODX
  • Сайты на WordPress
  • Сайты на платных и бесплатных темах Wordpress с кастомизацией, visual composer и другие
  • Правки по верстке, почта, JS(JQUERY), калькуляторы, wordpress, почта WP, MODX и т.д.

Стек технологий:

  • CMS Wordpress (посадка сайтов на WP)
  • CMS MODX (посадка сайтов на MODX)
  • PHP
  • HTML5, CSS3 (верстка сайтов по макетам PSD)
  • BOOTSTRAP
  • технология БЭМ и FLEX
  • препроцессоры LESS/SASS
  • JS (JQUERY)
  • технология AJAX
  • в проектах использую NPM, BOWER, GIT
  • сборщик проектов GULP
  • адаптивная и семантическая верстка
  • оптимизация скорость загрузки сайта (Google PageSpeed), валидация
  • Adobe Photoshop, Zeplin, Figma
